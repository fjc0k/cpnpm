# cpnpm

pnpm using npm.taobao.org as registry


## usage

```bash
npm install pnpm cpnpm -g
```

### install

same with `pnpm`, replace `pnpm` with `cpnpm` (or `tpm`), for example:

```bash
cpnpm add antd

# or
tpm add antd
```

same with `pnpx`, replace `pnpx` with `cpnpx` (or `tpx`), for example:

```bash
cpnpx kill-port 3000

# or
tpx kill-port 3000
```

### publish

use npm to publish
