# cpnpm

pnpm using npm.taobao.org as registry


## usage

```
npm install pnpm cpnpm -g
```

### install

same with `pnpm`, replace `pnpm` with `cpnpm`, for example:

```
cpnpm add antd
```

same with `pnpx`, replace `pnpx` with `cpnpx`, for example:

```
cpnpx kill-port 3000
```

### publish

use npm to publish
